Alpha Robot
Nuestro robot funciona con 3 motores cc, un servo motor y con un medidor de temperatura.
los motores cc son para la tracción y el servo para la dirección intentando imitar la dirección de un coche real. Con el sensor medimos la temperatura y cuando marque mas de 25 grados pararía a los motores cc. La dirección se controla mediante una app con el movil por bluetooth.
Como innovación seria ponerle una cámara al robot/coche para poder controlarlo por la app mas cómodamente.
